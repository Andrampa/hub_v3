# Handoff

## In Progress

The public catalog, country explorer, country detail pages, and OAuth with PKCE are implemented. End-to-end authentication completion still requires a human test with an enabled community account. Deployment selection and protected thematic group decisions remain open.

## Resume Checklist

1. Run `git status --short`.
2. Read `AGENTS.md` and `docs/context_index.md`.
3. Run `npm run build`.
4. For authentication work, read `docs/authentication.md` and inspect `src/services/auth.ts` first.
5. For country work, read `docs/country_explorer.md` and inspect `src/services/countries.ts` first.
